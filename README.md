# Kura Booru V2

Personal anime illustration collection and showcase platform.

**Core workflow**: Send a link via Telegram Bot → auto-download original image → store in S3 → browse on web.

Inspired by safebooru (tag system, pagination, fast loading) with modern UI (Pixiv/Pinterest-like masonry, auto/dark/light theme).

See [PLAN.md](./PLAN.md) for detailed architecture and development roadmap.
